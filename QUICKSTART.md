# 🚀 Быстрый старт

## Как запустить приложение

### Способ 1: Прямое открытие в браузере
1. Откройте файл `index.html` в вашем браузере
2. Приложение готово к использованию!

### Способ 2: Локальный сервер (рекомендуется)
Если вы хотите избежать проблем с CORS при работе с файлами:

**На macOS/Linux:**
```bash
cd driver-license-tracker
python3 -m http.server 8000
```

Затем откройте в браузере: `http://localhost:8000`

**На Windows:**
```bash
cd driver-license-tracker
python -m http.server 8000
```

Затем откройте в браузере: `http://localhost:8000`

## 📱 Первый запуск

1. **Посмотрите на шапку** - там огромный заголовок и дедлайн
2. **Найдите "Задачу на сегодня"** - три больших чекбокса для уроков
3. **Нажимайте на чекбоксы** - они будут светиться и издавать звуки!
4. **Смотрите прогресс** - глобальная карта обновляется автоматически

## 🎮 Основные функции

### Ежедневные задачи
- Отметьте 3 урока
- Решите билеты
- Получите конфетти при завершении всех уроков!

### Глобальная карта
- 88 кружочков = 88 шагов к экзамену
- Серые = еще не пройдены
- Яркие = уже пройдены
- Нет штрафов за пропуски!

### Инструменты для фокуса
- 🎵 **Lo-Fi плеер** - включите музыку для концентрации
- 🍅 **Помодоро** - 20 минут работы, 10 минут отдыха

### Кнопка спасения
- 🆘 **Не могу начать** - таймер на 5 минут с мотивацией

## 💾 Сохранение данных

Все ваши данные сохраняются автоматически в браузере (localStorage):
- ✅ Завершенные уроки
- ✅ Завершенные шаги
- ✅ Решенные билеты

**Важно:** Каждый новый день дневные задачи сбрасываются, но глобальный прогресс остается!

## 🔧 Отладка

В правом нижнем углу есть кнопка ⚙️ для отладки:
- 📥 Экспортировать данные (сохранить в JSON)
- 📤 Импортировать данные (загрузить из JSON)
- 📊 Показать статистику
- 🗑️ Очистить все данные

## 🌐 Браузеры

Приложение работает на всех современных браузерах:
- Chrome/Chromium
- Firefox
- Safari
- Edge

## 📞 Проблемы?

Если что-то не работает:
1. Откройте консоль браузера (F12)
2. Проверьте наличие ошибок
3. Очистите кэш браузера
4. Попробуйте другой браузер

## 🎯 Советы для максимальной эффективности

1. **Не смотрите на 88 сразу** - сосредоточьтесь на "Задаче на сегодня"
2. **Используйте Помодоро** - 20 минут легче, чем весь день
3. **Включите Lo-Fi** - музыка помогает концентрации
4. **Нажимайте чекбоксы** - дофамин от анимации мотивирует
5. **Не паникуйте о пропусках** - нет красных кружочков, нет стресса
