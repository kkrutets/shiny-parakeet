# 🎯 НАЧНИ ОТСЮДА

Добро пожаловать в **ПРОЕКТ: ВОДИТЕЛЬСКИЕ ПРАВА** 🚗💨

Это приложение создано специально для тебя, если у тебя СДВГ и ты готовишься к экзамену на водительские права.

## ⚡ Быстрый старт (30 секунд)

### Шаг 1: Открой приложение
Просто откройте файл `index.html` в браузере.

Или используйте локальный сервер:
```bash
python3 -m http.server 8000
# Затем откройте http://localhost:8000
```

### Шаг 2: Посмотри на экран
Ты увидишь:
- 🎯 Огромный заголовок "ПРОЕКТ: ВОДИТЕЛЬСКИЕ ПРАВА"
- 📊 Прогресс-бар сверху
- 📋 Три больших чекбокса для уроков
- 🗺️ Карту из 88 кружочков
- 🎵 Инструменты внизу

### Шаг 3: Нажми первый чекбокс
Услышь звук. Почувствуй вибрацию. Получи дофамин. 🎉

### Шаг 4: Повтори еще два раза
Три урока = три нажатия.

### Шаг 5: Смотри на конфетти
Когда все три урока готовы - ВЗРЫВ конфетти! 🎊

**Готово! Ты уже начал! 💪**

## 🎮 Основные функции

### 📋 Задача на сегодня
- 3 больших чекбокса для уроков
- 1 чекбокс для решения билетов
- Звуки, вибрация, конфетти при завершении

### 🗺️ Глобальная карта: 88 шагов
- Визуальная сетка из 88 кружочков
- Серые = еще не пройдены
- Яркие = уже пройдены
- Нет штрафов за пропуски!

### 🆘 Кнопка спасения
- Для дней, когда не можешь начать
- Таймер на 5 минут
- Мотивирующее сообщение

### 🎵 Lo-Fi плеер
- Встроенная музыка для концентрации
- Одна кнопка для включения

### 🍅 Помодоро таймер
- 20 минут работы
- 10 минут отдыха
- Идеально для СДВГ

## 🧠 Почему это работает для СДВГ

### ❌ Проблема: Исполнительная дисфункция
✅ **Решение:** Микро-задачи вместо больших целей
- Вместо "подготовься к экзамену" → "сделай 3 урока сегодня"

### ❌ Проблема: Дефицит внимания
✅ **Решение:** Визуальные стимулы и звуки
- Яркие цвета, звуки, вибрация, конфетти

### ❌ Проблема: Импульсивность
✅ **Решение:** Немедленное вознаграждение
- Нажал → сразу звук и вибрация

### ❌ Проблема: Прокрастинация
✅ **Решение:** Кнопка "Не могу начать"
- 5 минут - это управляемо

### ❌ Проблема: Чувство вины
✅ **Решение:** Отсутствие штрафов
- Нет красных кружочков, нет "просрочек"

## 🎯 Советы для максимальной эффективности

1. **Не смотри на 88 сразу** - сосредоточься на "Задаче на сегодня"
2. **Используй Помодоро** - 20 минут легче, чем весь день
3. **Включи Lo-Fi** - музыка помогает концентрации
4. **Нажимай чекбоксы** - дофамин от анимации мотивирует
5. **Не паникуй о пропусках** - нет красных кружочков, нет стресса
6. **Используй кнопку спасения** - когда не можешь начать
7. **Смотри на прогресс** - видимый прогресс мотивирует
8. **Делай перерывы** - Помодоро помогает избежать выгорания

## 🆘 Проблемы?

### Приложение не открывается
- Убедитесь, что файл `index.html` существует
- Попробуйте открыть в другом браузере
- Используйте локальный сервер

### Звуки не работают
- Проверьте громкость браузера
- Проверьте разрешения на звук
- Попробуйте другой браузер

### Данные не сохраняются
- Убедитесь, что localStorage включен
- Не используйте приватный режим браузера
- Очистите кэш браузера

### Вибрация не работает
- Вибрация работает только на мобильных устройствах
- Убедитесь, что вибрация включена в настройках
