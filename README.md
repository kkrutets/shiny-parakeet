# 🚗 ПРОЕКТ: ВОДИТЕЛЬСКИЕ ПРАВА 💨

Интерактивный трекер прогресса подготовки к экзамену на водительские права, специально разработанный для людей с СДВГ.

## 🎯 Основные возможности

### 1. **Шапка с мотивацией**
- Огромный заголовок проекта
- Главный прогресс-бар (0% → 100%)
- Дружелюбный дедлайн без паники

### 2. **Блок "Задача на сегодня"**
- 3 больших чекбокса для уроков
- 1 чекбокс для решения билетов
- **Магия дофамина**: 
  - Яркая анимация при клике ✨
  - Звук "ПУНЬК!" 🔊
  - Вибрация телефона 📳
  - Конфетти при завершении всех уроков 🎉

### 3. **Глобальная карта: 88 шагов**
- Визуальная сетка из 88 кружочков
- Серые кружочки = еще не пройдены
- Яркие кружочки = уже пройдены
- **Без штрафов за пропуски!** Нет красных кружочков, нет паники

### 4. **Кнопка спасения**
- 🆘 "Не могу начать"
- Открывает модальное окно с таймером на 5 минут
- Мотивирующее сообщение: "Просто включи видео на х1.5 и посмотри 5 минут"

### 5. **Инструменты для фокуса**
- 🎵 **Lo-Fi плеер**: Встроенный плеер для концентрации
- 🍅 **Помодоро таймер**: 20 минут работы / 10 минут отдыха

## 🚀 Как использовать

### Установка
1. Откройте файл `index.html` в браузере
2. Все данные сохраняются автоматически в localStorage

### Ежедневный процесс
1. **Утром**: Посмотрите на "Задачу на сегодня"
2. **Во время учебы**: Нажимайте на чекбоксы уроков
3. **После уроков**: Отметьте решение билетов
4. **Смотрите прогресс**: Глобальная карта обновляется автоматически

### Если застрял
- Нажмите 🆘 "Не могу начать"
- Включите Lo-Fi музыку
- Используйте Помодоро таймер (20 минут = легко!)

## 💾 Сохранение данных

Все ваши прогресс сохраняется автоматически:
- ✅ Завершенные уроки
- ✅ Завершенные шаги на карте
- ✅ Решенные билеты

**Важно**: Каждый новый день дневные задачи (уроки и билеты) сбрасываются, но глобальный прогресс остается!

## 🎵 Звуки

Приложение использует Web Audio API для создания звуков:
- **Punch Sound**: При клике на чекбокс (800 Hz)
- **Victory Sound**: При завершении всех уроков (Do-Mi-Sol)
- **Startup Sound**: При загрузке страницы

## 💡 Советы для максимальной эффективности

1. **Не смотрите на 88 сразу** - смотрите только на "Задачу на сегодня"
2. **Используйте Помодоро** - 20 минут легче, чем "весь день"
3. **Включите Lo-Fi** - музыка помогает концентрации
4. **Нажимайте чекбоксы** - дофамин от анимации мотивирует
5. **Не паникуйте о пропусках** - нет красных кружочков, нет стресса
