## 📄 Описание каждого файла

### [`index.html`](index.html)
**Основная разметка приложения**

Содержит:
- Шапка с заголовком и прогресс-баром
- Блок "Задача на сегодня" с чекбоксами
- Глобальная карта из 88 кружочков
- Кнопка спасения
- Подвал с инструментами (Lo-Fi плеер, Помодоро)
- Модальное окно для кнопки спасения
- Отладочное меню

**Размер:** ~3 KB
**Зависимости:** config.js, debug.js, adhd-features.js, script.js

---

### [`styles.css`](styles.css)
**Стили и CSS анимации**

Содержит:
- CSS переменные для цветов и размеров
- Стили для всех компонентов
- Анимации (slideDown, checkPunch, confettiFall, pulse и т.д.)
- Адаптивный дизайн для мобильных устройств
- Стили для отладочного меню
- Градиенты и тени

**Размер:** ~15 KB
**Особенности:**
- Использует CSS Grid и Flexbox
- Поддерживает темный режим (опционально)
- Полностью адаптивен

---

### [`script.js`](script.js)
**Основная логика приложения**

Содержит:
- Управление состоянием приложения
- Инициализацию сетки 88 шагов
- Обработчики событий для чекбоксов
- Обновление прогресс-бара
- Логику уроков на сегодня
- Систему конфетти
- Модальное окно спасения
- Lo-Fi плеер
- Помодоро таймер
- Синтез звуков (Web Audio API)
- Сохранение/загрузку данных (localStorage)
- Обновление дедлайна

**Размер:** ~20 KB
**Функции:**
- `initializeStepsGrid()` - создание сетки
- `toggleStep()` - переключение шага
- `updateProgressBar()` - обновление прогресса
- `toggleLesson()` - переключение урока
- `triggerConfetti()` - запуск конфетти
- `playPunchSound()` - звук клика
- `playVictorySound()` - звук победы
- И многие другие...

---

### [`config.js`](config.js)
**Конфигурация и утилиты**

Содержит:
- Объект CONFIG с параметрами приложения
- Объект UTILS с вспомогательными функциями
- Параметры звуков и вибрации
- Параметры конфетти
- Сообщения приложения
- Функции для работы с localStorage
- Функции для экспорта/импорта данных
- Функции для получения статистики

**Размер:** ~8 KB
**Основные параметры:**
- `EXAM_DATE` - дата экзамена
- `TOTAL_STEPS` - количество шагов (88)
- `LESSONS_PER_DAY` - уроков в день (3)
- `POMODORO_WORK` - 20 минут
- `POMODORO_BREAK` - 10 минут
- `RESCUE_TIMER` - 5 минут

## 🔄 Зависимости между файлами

```
index.html
├── config.js (подключается первым)
├── debug.js (подключается вторым)
├── adhd-features.js (подключается третьим)
└── script.js (подключается последним)
    ├── использует CONFIG из config.js
    ├── использует UTILS из config.js
    ├── использует ADHDFeatures из adhd-features.js
    └── использует функции из debug.js

styles.css
└── используется в index.html

Документация:
├── README.md (полная документация)
├── QUICKSTART.md (быстрый старт)
├── ADHD-GUIDE.md (для СДВГ)
├── PROJECT-STRUCTURE.md (этот файл)
└── package.json (информация о проекте)
```

## 🚀 Порядок загрузки

1. **HTML парсится** → index.html
2. **CSS загружается** → styles.css
3. **JavaScript загружается в порядке:**
   - config.js (конфигурация)
   - debug.js (отладка)
   - adhd-features.js (СДВГ функции)
   - script.js (основная логика)
4. **DOM готов** → DOMContentLoaded событие
5. **Инициализация** → setupDebugMenu(), initializeStepsGrid() и т.д.

## 💾 Хранение данных

Все данные хранятся в `localStorage` браузера:
- Ключ: `driverLicenseTracker`
- Формат: JSON
- Содержит:
  - `completedStepsArray` - массив завершенных шагов
  - `todayLessons` - уроки на сегодня
  - `ticketsCompleted` - решены ли билеты
  - `lastSaveDate` - дата последнего сохранения

## 🔧 Как расширить проект

### Добавить новую функцию
1. Добавить функцию в `script.js` или `adhd-features.js`
2. Добавить обработчик события в `setupEventListeners()`
3. Добавить стили в `styles.css`
4. Обновить документацию

### Изменить конфигурацию
1. Отредактировать `config.js`
2. Обновить `package.json`
3. Обновить документацию

### Добавить новый звук
1. Добавить функцию в `script.js`
2. Использовать Web Audio API
3. Добавить параметры в `config.js`
