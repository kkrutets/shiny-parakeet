## 🚀 Установка и запуск

### Шаг 1: Установить зависимости

```bash
cd driver-license-tracker
npm install --save-dev @playwright/test
```

### Шаг 2: Убедиться, что сервер работает

```bash
# В отдельном терминале
cd driver-license-tracker
python3 -m http.server 8000
```

### Шаг 3: Запустить тесты

```bash
# Запустить все тесты
npx playwright test

# Запустить тесты в режиме просмотра
npx playwright test --ui

# Запустить тесты в конкретном браузере
npx playwright test --project=chromium

# Запустить конкретный тест
npx playwright test tests/app.spec.js -g "должна отображать заголовок"

# Запустить тесты с отчетом
npx playwright test
npx playwright show-report
```

## 🔧 Конфигурация

### Файл `playwright.config.js`

Содержит:
- Директорию с тестами: `./tests`
- Браузеры для тестирования: Chromium, Firefox, WebKit
- Мобильные устройства: Pixel 5, iPhone 12
- Базовый URL: `http://localhost:8000`
- Веб-сервер: Python HTTP server на порту 8000

### Изменение конфигурации

Если вы хотите изменить:
- **Браузеры**: отредактируйте секцию `projects`
- **Базовый URL**: измените `baseURL` в `use`
- **Порт сервера**: измените `url` в `webServer`
- **Таймауты**: добавьте `timeout` в `use`

## 📝 Написание новых тестов

### Шаблон теста
```javascript
test('описание теста', async ({ page }) => {
  // Перейти на страницу
  await page.goto('http://localhost:8000');
  
  // Найти элемент
  const element = await page.locator('.selector');
  
  // Проверить видимость
  await expect(element).toBeVisible();
  
  // Кликнуть на элемент
  await element.click();
  
  // Проверить текст
  await expect(element).toContainText('текст');
});
```

### Полезные методы
```javascript
// Навигация
await page.goto('url');
await page.reload();
await page.goBack();

// Поиск элементов
await page.locator('selector');
await page.getByText('текст');
await page.getByRole('button', { name: 'Нажми' });

// Взаимодействие
await element.click();
await element.fill('текст');
await element.type('текст');
await element.press('Enter');

// Проверки
await expect(element).toBeVisible();
await expect(element).toContainText('текст');
await expect(element).toHaveClass('класс');
await expect(element).toHaveAttribute('атрибут', 'значение');

// Ожидание
await page.waitForLoadState('domcontentloaded');
await page.waitForSelector('selector');
await page.waitForTimeout(1000);
```

## 🚨 Частые проблемы

### Проблема: Тесты не находят элементы
**Решение:**
1. Проверьте селектор в консоли браузера
2. Убедитесь, что элемент видимый
3. Используйте `page.pause()` для отладки

### Проблема: Сервер не запускается
**Решение:**
1. Убедитесь, что порт 8000 свободен
2. Запустите сервер вручную перед тестами
3. Измените порт в `playwright.config.js`

### Проблема: Тесты зависают
**Решение:**
1. Увеличьте таймаут в конфигурации
2. Добавьте явное ожидание элемента
3. Используйте `--debug` для отладки

### Проблема: localStorage не работает в тестах
**Решение:**
1. Используйте `page.evaluate()` для доступа к localStorage
2. Убедитесь, что браузер поддерживает localStorage
3. Очистите localStorage перед тестом
